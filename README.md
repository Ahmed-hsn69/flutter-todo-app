# To-Do Plus

## Overview

To-Do Plus is a Flutter-based mobile application developed to improve productivity by combining task management, digital sketching, and voice recording into a single platform. Instead of relying on multiple applications, users can manage daily tasks, create drawings, and record audio notes within one convenient and user-friendly environment.

## Features

### Task Management
- Create, edit, and delete tasks
- Mark tasks as completed
- Organize and track daily activities

### Doodle Canvas
- Freehand drawing functionality
- Multiple color options and adjustable brush sizes
- Undo and clear canvas tools
- Save drawings to the device gallery
- Firebase backup support

### Voice Recorder
- Record and store audio notes
- Play, pause, and seek recordings
- Rename and delete recordings
- Local storage with Firebase synchronization

## Technologies Used

- Flutter
- Dart
- Firebase Firestore
- Flutter Sound
- Local Storage
- Android Gallery Integration

## Purpose

The goal of To-Do Plus is to provide a simple and efficient solution for productivity, creativity, and note-taking. By integrating task management, drawing, and audio recording into a single application, users can easily organize information, capture ideas, and manage their daily activities.
